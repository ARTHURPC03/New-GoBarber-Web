## 💻 Projeto

GoBarber é um aplicativo que permite aos usuários agendar uma consulta com seu barbeiro favorito e mostra ao barbeiro sua agenda para o dia.

## ⚙ Configuração

1- Para instalar as dependências:
> yarn

2- Para iniciar a aplicação:
> yarn start
